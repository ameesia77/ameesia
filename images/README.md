## Image Naming Convention

For best results, name your images like this:

### Cover Images (for the grid):
- `cover.jpg` - Main image for the grid
- `cover-alt.jpg` - Alternative if needed

### Gallery Images (for project pages):
- `01.jpg`, `02.jpg`, `03.jpg` etc.
- Or descriptive: `opening-night.jpg`, `installation-view.jpg`

## Image Specifications

**Recommended Sizes:**
- Cover images: 1200x800px or 800x1200px (landscape/portrait)
- Gallery images: 1920px wide max
- File format: JPG (for photos), PNG (for graphics/text)
- Keep file sizes under 2MB for web performance

## How to Add Images

1. **Drop images into the appropriate folders:**
   - Example: Solienne images → `images/automata/solienne/`
   - Example: Venice photos → `images/bright-moments/venice/`

2. **Name the main grid image `cover.jpg`**

3. **Name additional images numerically or descriptively**

## Quick Drop Zone

If you're unsure where something goes, drop it in `images/misc/` and we'll organize it together.
